---
title: "List favorites"
description: "Get the authenticated user's list of favorited recipes."
---

<Warning>
  This endpoint requires authentication. Requests without a valid session return `401 No autorizado`.
</Warning>

## Endpoint

```
GET /api/favoritos
```

## Request

No request body or query parameters required. The user is identified from the active session cookie.

## Response

Returns an array of recipe objects. Each item is a full recipe record with the author's profile name joined.

<ResponseField name="[]">Array of favorited recipe objects, ordered by the date they were saved (most recent first).</ResponseField>

<ResponseField name="[].id" type="string">
  Unique identifier of the recipe (UUID).
</ResponseField>

<ResponseField name="[].titulo" type="string">
  Title of the recipe.
</ResponseField>

<ResponseField name="[].descripcion" type="string">
  Short description of the recipe.
</ResponseField>

<ResponseField name="[].imagen_url" type="string">
  Public URL of the recipe's cover image.
</ResponseField>

<ResponseField name="[].usuario_id" type="string">
  UUID of the user who created the recipe.
</ResponseField>

<ResponseField name="[].perfiles" type="object">
  Author profile data joined from the `perfiles` table.
  <Expandable title="properties">
    <ResponseField name="nombre" type="string">
      Display name of the recipe's author.
    </ResponseField>
  </Expandable>
</ResponseField>

## Error responses

| Status | Error | Description |
|--------|-------|-------------|
| `401` | `No autorizado` | No active session or invalid session token. |
| `400` | Database error message | A Supabase query error occurred. |
| `500` | `Error interno` | An unexpected server error occurred. |

<RequestExample>

```bash cURL
curl --request GET \
  --url https://your-app.vercel.app/api/favoritos \
  --header 'Cookie: sb-access-token=<your-session-token>'
```

</RequestExample>

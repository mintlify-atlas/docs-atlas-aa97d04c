## Prerequisites

Before you start, make sure you have the following installed:

- **Node.js** v18 or later — [nodejs.org](https://nodejs.org)
- **npm** (comes with Node.js)
- **Git**
- A **Supabase** account — [supabase.com](https://supabase.com) (free tier works)

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/alcorher/entregable3_di.git
    cd entregable3_di
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```

    This installs Next.js, React, the Supabase client libraries, and TailwindCSS.
  </Step>

  <Step title="Create your environment file">
    Create a `.env.local` file in the project root with your Supabase credentials:

    ```env .env.local
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

    <Warning>
      Never commit `.env.local` to version control. It contains sensitive keys. The `.gitignore` already excludes it.
    </Warning>

    To find your credentials, go to your [Supabase project dashboard](https://supabase.com/dashboard), open **Settings → API**, and copy the **Project URL** and **anon public** key.

    See [Environment Setup](/environment-setup) for a full walkthrough.
  </Step>

  <Step title="Set up the database">
    Your Supabase project needs the `perfiles`, `recetas`, and `favoritos` tables. Follow the [Supabase Setup guide](/database/supabase-setup) to run the SQL DDL and configure storage buckets.
  </Step>

  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    <Tip>
      Next.js supports hot reloading — any changes you save will instantly reflect in the browser without a full restart.
    </Tip>
  </Step>

  <Step title="Open the app">
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser. You should see the Sazón Comunitario landing page.
  </Step>

  <Step title="Create your account">
    Click **Regístrate** on the landing page, fill in your email, password, and display name, then log in. You're now ready to browse recipes, upload your own, and explore the platform.

    <Note>
      The app ships with a sample admin account. If you're connecting to the shared database, you can log in with `admin@gmail.com` / `123456` for admin-level access.
    </Note>
  </Step>
</Steps>

## Available scripts

| Script | Command | Description |
|--------|---------|-------------|
| Development | `npm run dev` | Starts Next.js dev server with hot reload on port 3000 |
| Build | `npm run build` | Creates an optimized production build |
| Start | `npm start` | Serves the production build locally |
